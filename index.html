Estimated Arrival July 25, 2020

Learn to Code with Python is a *free* comprehensive programming course. It's designed for the newbie: never written a line of code before? No worries. I've got you.

This course, and the website and community that I hope to build around it, is especially designed for people who are underrepresented in tech: women, queer folk, BIPOC.

I want to create an online space where diversity is centered and celebrated. Here, it will be safe for everyone to learn and ask questions. If software engineering and tech are interesting to you, let this be the place you begin your programming journey.

See content outline and progress updates.
