Learn to Code with Python is a free comprehensive programming course. It's designed for the newbie: never written a line of code before? No worries. I've got you.

This course, and the website and community that I hope to build around it, is for people who are underrepresented in tech.

I want to create an online space where diversity is centered and celebrated. Here, it will be safe for everyone to learn and ask questions. If you are interested in software engineering and tech, you are welcome here. Let this be the place you begin your programming journey.
